## Module Structure

The AI Detection module is located at `/ai_detection/` and includes:

- **SPAI** - Spectral AI-Generated Image Detector (CVPR 2025)
- **SDXL Detector** - Swin Transformer-based detector for SDXL-generated images
- **Plugin System** - Extensible detector architecture
